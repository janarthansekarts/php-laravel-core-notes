# PHP & Laravel Core Notes

This repository contains my personal learning notes while deeply studying **PHP Object-Oriented Programming** and **Laravel framework internals**.

These are not tutorial notes or documentation copies.  
They are written to build **conceptual clarity** around *why things exist* and *how they work internally*, from the perspective of a working software engineer.

The focus is on:
- PHP OOP fundamentals (from first principles)
- Laravel architecture and internals
- How enterprise frameworks are designed
- Connecting language concepts to real-world framework behavior

---

## Who this is for

- Me (primary): a long-term reference and second brain  
- Other developers who want deeper understanding beyond syntax  

If you’re looking for “how to build CRUD apps” or quick start guides, this repo is **not** for that.
